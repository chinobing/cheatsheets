## 晶圆
{: .-two-column}

### 尺寸

| 英寸 _(inch)_   | 直径 _(mm)_   |
| --------------- | ------------ |
| `1-inch`        | 25 mm        |
| `4-inch`        | 100 mm        |
| `5-inch`        | 125 mm        |
| `6-inch`        | 150 mm        |
| `8-inch`        | 200 mm        |
| `12-inch`       | 300 mm        |

注：基本是用 **inch 的大小** 乘以 **25mm** 可以得出**晶圆尺寸**


### 等效换算

#### 公式
- $ 面积 = \pi r^2 $

#### 4寸(100mm) 转 8寸(200mm)
- 4寸面积 = 7853mm
- 8寸面积 = 31415mm
- 4寸的产能为 4万片，那么8寸的等效产能为 **7853mm/31415mm = 0.25 * 4万片 = 1万片 8寸产能**
